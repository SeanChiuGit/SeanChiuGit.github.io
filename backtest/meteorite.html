Meteorite Analysis Dashboard

Python, Plotly, HTML, Data Science

This project analyzes NASA's open meteorite dataset using geospatial and statistical techniques. The goal is to identify landing patterns, meteorite types, and size distributions around the world.

The dashboards below are built with Plotly and show:

  • 🌍 Global meteorite landing map
  • 📊 Composition distribution
  • 🪨 Mass and classification visualizations
